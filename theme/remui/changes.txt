Release
=======

Version 4.5.12
=============
Tweak - Adjusted the Focus Mode toast message position slightly downward for better visibility of the course name and navigation buttons.
Fix - Resolved minor UI issues in the theme.
Fix - Corrected an issue where dashboard stats were not updating when a course was hidden.
Fix - Fixed the file picker modal not working on the latest version of Moodle.
Fix - Fixed the login page site description color not updating when the color palette was changed.
Fix - Fixed login page colors not updating when Smart Color was applied.

Version 4.5.11
=============
Feature - Added 6 footer design templates in the Visual Personalizer.
Feature - Added an option to download footer settings in JSON format to back up the footer configuration.
Feature - Added separate background color and text color settings for the main footer area and bottom footer area in footer templates 1 and 2.
Tweak - Added background image settings in all footer templates except the default footer.
Tweak - Added logo settings in the widget area.
Tweak - Added an email form and its settings in footer templates 2, 3, 4, and 6.
Tweak - Moved footer social icon settings to the main area for the default and template 1, and to the bottom area for other templates.
Tweak - Moved footer bottom text and link settings to the bottom area in the default footer settings.
