# BitAscii Solutions #

Create contextful pages for your Moodle site.

It helps you create contextful pages which can have blocks added in it(blocks which allows to be added to all layout). Pages can be restricted to certain audiences like cohort, certain roles, all users or can be given access to certain users only. You can also allow guest to visit pages which you think should be accessible by un-authorised users.

## Installing via uploaded ZIP file ##

1. Log in to your Moodle site as an admin and go to _Site administration >
   Plugins > Install plugins_.
2. Upload the ZIP file with the plugin code. You should only be prompted to add
   extra details if your plugin type is not automatically detected.
3. Check the plugin validation report and finish the installation.

## Installing manually ##

The plugin can be also installed by putting the contents of this directory to

    {your/moodle/dirroot}/local/iomadcustompage

Afterwards, log in to your Moodle site as an admin and go to _Site administration >
Notifications_ to complete the installation.

Alternatively, you can run

    $ php admin/cli/upgrade.php

to complete the installation from the command line.
